# Synthesio

**Paint music into existence.**

A browser-based canvas that converts your painting into a live musical composition in real time — no AI APIs, no servers, no install. Open the file and it works.

Built for [Hack The Arts 2026](https://devpost.com/) · Theme: *"Create art that couldn't exist without technology."*

---

## How it works

Draw anything on the canvas. The moment you hit **Analyze & Play**, a custom pixel analysis engine reads the raw image data — color distribution, edge texture, brightness, quadrant composition — and maps it directly to musical parameters. Tone.js turns those parameters into a live synthesizer performance that evolves as you keep painting.

The music isn't generated by an AI model. It's computed mathematically from your brushstrokes.

---

## What it does

**Painting → Music pipeline**

- Average hue maps to musical key via the circle of fifths, weighted by saturation
- Edge density drives tempo and articulation (smooth washes = slow, heavy strokes = fast)
- Color variance selects the scale: pentatonic for tight palettes, chromatic for wild ones
- Brightness controls reverb depth and sustain
- Quadrant analysis feeds per-zone note weighting — what you paint where matters

**Musical engine**

- Main melody voice with quadrant-weighted note selection
- Counterpoint voice on off-beats
- Chord mode on sparse, low-density paintings (full triads instead of single notes)
- 4 arpeggiation patterns that switch automatically based on saturation and density
- Bass voice on dark or bass-instrument paintings
- Auto-reanalysis every 6 seconds while playing — music shifts as you keep painting, with smooth BPM and reverb transitions (no audio restarts)

**While playing, painting becomes an instrument**

- Every brushstroke fires an immediate note — Y position maps to pitch, so drawing high triggers high notes
- The waveform visualizer shows 3 modes: sine wave, bar chart, or particle field

**Everything else**

- Live brush cursor (exact size, color, opacity)
- Color palette tooltip shows the musical key and mood each color maps to
- Tempo tap button to manually sync BPM by tapping
- Canvas zoom/pan (mouse wheel + alt-drag), painting timer, undo history with counter
- Export the canvas as PNG, download the composition as a MIDI file, or export a human-readable score analysis as text
- Share button encodes the painting into the URL hash so anyone with the link sees the same canvas and parameters

## Stack

| Layer | Tool |
|---|---|
| Canvas + UI | Vanilla HTML5 Canvas, CSS Grid, no framework |
| Pixel analysis | Custom JS — `analyzeCanvas()` and helpers |
| Music synthesis | [Tone.js 14.8.49](https://tonejs.github.io/) |
| MIDI export | [midi-writer-js 2.1.4](https://github.com/grimmdude/MidiWriterJS) |
| Fonts | Space Mono + Inter (Google Fonts) |
| Hosting | GitHub Pages |

No build step. One file.
